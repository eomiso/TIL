# 해결방안 2: Numpy의 Test Support를 사용하기

Test Support (numpy.testing)

그래서 구글링을 하다보니 numpy.testing이라는 라이브러리를 알게 되었다. numpy안에 포함된 Assert들의 모음집인데, 테스트를 통과하지 못하는 경우에는 AssertionError를 일으킨다.

1.39770684 로 화면에 나타나는 값이 실제로는 1.3977068363854122 이어서 테스트를 통과하지 못하는 경우가 있다. 이런 경우에는 assert_Almost_Equal을 사용해보자.
